# 🛒 E-Commerce Store [Frontend] (AngularJS)

A responsive and dynamic single-page e-commerce web application built with **AngularJS**. This project simulates a basic online shopping platform, allowing users to browse products, add items to a cart, and place orders.

---

## 🚀 Features

- 📦 Product listing with filtering and categorization
- 🛍️ Shopping cart functionality
- 🔍 Product detail pages
- 🧾 Order summary and checkout simulation
- 🎨 Responsive UI/UX with modern styling
- 🔄 Dynamic routing with AngularJS
- 🧩 Modular code structure for scalability

---

## 🛠️ Installation & Setup

### Prerequisites

- [Node.js](https://nodejs.org/) >= 18.x
- [Angular CLI](https://angular.io/cli) >= 19.2.11
- Git

### Clone & Install

```bash
git clone git@github.com:VikrantParmar/ecommerce-store-angularjs.git
cd ecommerce-store-angularjs
npm install
````

### Run the Development Server

```bash
ng serve
```

Visit the app in your browser:
📍 [http://localhost:4200](http://localhost:4200)

## 📦 Future Enhancements

Here's a roadmap of features planned to enhance the functionality, user experience, and scalability of the platform:

### 🛒 Shopping Experience

- ❤️ **Wishlist Functionality**  
  Allow users to save favorite items and access them later.
  
- 🌟 **Featured Products Section**  
  Highlight curated or promoted products on the homepage.

- ⚡ **Flash Deals / Limited-Time Offers**  
  Introduce countdown timers and flash discounts to boost sales.

### 🧾 Orders & Transactions

- 🧾 **Invoice Generation (PDF)**  
  Generate downloadable invoices on order confirmation and send via email.

- 💬 **SMS Gateway Integration**  
  Send real-time order confirmations, delivery updates, and alerts via SMS.

- ♻️ **Refund & Return Process**  
  Provide UI for initiating and managing refund or return requests.

### 🔐 User Experience

- 🌐 **Multi-Language & Currency Support**  
  Localize site content and pricing for international users.

- 🛡️ **Enhanced Security Features**  
  CAPTCHA on forms and session expiration.
